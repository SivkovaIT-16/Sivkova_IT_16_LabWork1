Дана сигнатура метода: public int[] findAll (int[] arr, int x);
Необходимо реализовать метод таким образом, чтобы он возвращал новый
массив, в котором записаны индексы всех вхождений числа x в массив arr.
Пример:
arr=[1,2,3,8,2,2,9]
x=2
результат: [1,4,5]
### Алгоритм решения
Метод findAll, расположенный в Lab1,  принимает на вход массив arr типа int[] и искомое число x типа int. Сначала подсчитывается количеств вхождений числа x в массив. Для этого: Инициализируется счетчик count со значением 0, цикл последовательно перебирает все элементы массива, при каждом совпадении arr[i] == x счетчик увеличивается на 1. Далее создается массив res размером count (количество найденных вхождений). Второй цикл снова перебирает все элементы массива. При каждом совпадении индекс i записывается в результирующий массив. Метод возвращает массив res с индексами всех вхождений числа x в исходный массив. В основном коде, перед вызовом метода findAll пользователь вводит размерность массива (проверяется на корректность с помощью приватных методов xIsNatural и MasSize), элементы массива и число x.
